# Smart Multimodal AI Chatbot

## Short Description
A professional AI chatbot with multimodal support that can process **text, image, audio, and video inputs**, handle long media efficiently, and respond with multiple personalities and emoji support. Built using **Hugging Face models, Transformers, and Gradio**.

## About the Project
The **Smart Multimodal AI Chatbot** is an advanced conversational AI designed for both fun and productivity. Users can chat via text or upload media files (image, audio, or video), and the bot will generate intelligent, context-aware responses.

**Key Highlights:**
- **Multimodal input:** Single interface supports text, image, audio, or video.
- **Personality modes:** 10 selectable personalities for varied response styles.
- **Long-media handling:** Automatic transcription and summarization for long audio/video files.
- **Professional UI:** Modern dark-mode interface with floating media preview and styled chat bubbles.
- **Emoji support:** Optional emoji replacement for a more engaging chat experience.
- **Open-source friendly:** Built using Python, Transformers, Hugging Face, and Gradio.

## Features
- Text, image, audio, and video input in one unified UI.
- Progressive summarization of long audio/video to maintain context.
- Multiple chatbot personalities for tailored responses.
- Emoji-enabled messages for a more engaging conversation.
- Dark-mode UI with professional design and modern chat bubbles.
- Clear chat and adjustable model parameters (temperature, max tokens).
